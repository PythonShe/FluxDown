本文带你走完在 FluxDown 中新建下载、控制运行中任务、用队列与限速组织任务的完整流程。如果还没安装,先看[安装](/docs/zh/getting-started/installation/);想了解各面板的具体位置,可参考[界面总览](/docs/zh/getting-started/interface/)。

## 创建第一个下载任务

点击顶栏的**新建下载**(或按 <kbd>Ctrl</kbd>/<kbd>Cmd</kbd>+<kbd>F</kbd> 搜索后从结果中打开)。对话框字段自上而下依次为:

| 字段 | 作用 |
|---|---|
| **下载链接** | 多行输入框——每行粘贴一个链接即可批量下载(磁力链接与 `ed2k://` 链接同样支持),单条链接则只建一个任务。输入框上方会实时显示已解析的链接数。**打开种子文件**按钮可选择本地 `.torrent`,**导入 TXT 文件**按钮可从文本文件批量导入链接列表。 |
| **保存目录** | 文件保存位置。默认取全局保存目录(**设置 → 下载**),若开启了"跟随上次保存位置"则取上次使用的目录。 |
| **线程数** | 分片下载的线程数:**自动**(FluxDown 按文件大小和 CPU 核心数动态计算)、预设值(4/8/16/32/64),或 1–256 之间的自定义值。磁力链接和 `.torrent` 文件不显示此项,因为 BT 引擎自行管理连接数。 |
| **重命名(可选)** | 覆盖自动识别的文件名。仅单条链接时显示——批量下载和种子任务始终使用识别到的原始文件名。 |
| **队列** | 将任务分配到某个命名队列而非默认队列。只有创建过至少一个命名队列后才会显示(见下文)。 |

点击**高级选项**展开逐任务覆盖项,留空则沿用全局设置:

| 字段 | 作用 |
|---|---|
| **任务代理** | 仅用于此任务的代理,例如 `socks5://127.0.0.1:1080` 或 `http://host:port`。留空则使用全局代理。BT 下载不支持代理。 |
| **User-Agent** | 选择预设(Chrome、Firefox、Edge、Safari),或直接输入自定义字符串。 |
| **Cookie** | `name=value; name2=value2` 格式的原始 Cookie,用于需要登录态的下载。 |
| **哈希校验** | 选择算法(MD5/SHA-1/SHA-256/SHA-512)并粘贴期望的哈希值,下载完成后 FluxDown 会校验文件完整性,不一致时给出提示。留空则跳过校验。 |
| **自定义请求头** | 以键/值行形式附加额外 HTTP 请求头(点击**添加请求头**可增加更多行)。Cookie 请使用上方独立的 Cookie 字段,而不是在这里填 `Cookie` 请求头。 |

点击**开始下载**(批量时显示为**下载 N 个文件**)即可开始。

## 任务控制

- **双击**任务行:进行中的任务会暂停,已暂停/失败的任务会恢复,已完成的任务则直接打开文件。
- **右键**任务行弹出完整菜单:暂停/继续、优先下载(见下文)、打开文件(仅已完成任务)/打开所在文件夹、复制下载地址、删除任务、删除任务和文件。
- **批量操作**:按 <kbd>Ctrl</kbd>/<kbd>Cmd</kbd>+<kbd>A</kbd>,或点击文件名列旁的管理模式切换按钮,逐个勾选任务后用批量操作栏删除(可选是否连同文件一起删除)。按 <kbd>Esc</kbd> 退出管理模式。
- **全部暂停/恢复**:使用顶栏的全部暂停/全部恢复按钮,或在列表下方空白处右键选择全部开始/全部暂停。

## 队列与限速入门

窗口底部**状态栏**提供全局限速:点击限速指示区打开弹出面板,可选预设值(128 KB/s、512 KB/s、1 MB/s、2 MB/s、5 MB/s)或输入自定义 KB/s 数值,默认关闭(不限速)。

需要更精细的控制,可在侧边栏**队列**区块点击 **+** 创建**命名队列**。每个队列拥有独立的:

- 速度限制(KB/s,0 = 不限制)
- 最大同时下载数(0 = 使用全局设置)
- 默认保存目录
- 默认线程数(0 = 自动)
- 默认 User-Agent(留空则继承全局设置)

在新建下载对话框的队列下拉框中把任务分配到某个队列。在侧边栏悬浮或右键某个队列可编辑或删除它——删除队列会将其中的任务移回默认队列。点击侧边栏的状态或队列条目即可按其筛选任务列表。

全局**最大同时下载数**上限(默认 5)位于**设置 → 下载**,对所有未单独设置该项的队列生效。

## Boost:优先下载

急需某个文件时,右键任意未完成的任务并选择**优先下载**。FluxDown 会暂停其他所有进行中的任务,把带宽全部让给该任务;任务列表顶部会出现提示条,显示完成后将自动恢复的任务数,并带有**取消**链接可随时终止。被优先下载的任务行上还会出现一个小闪电徽标。再次右键该任务选择**取消优先**即可手动解除。

## 下载完成后

默认情况下(**设置 → 通用 → 完成通知**),FluxDown 会弹出带**打开文件**/**打开所在文件夹**操作按钮的系统通知——Windows 上是右下角的悬浮提示窗,macOS/Linux 上是系统原生通知。短时间内多个任务同时完成时,通知会自动合并显示。你也可以随时双击已完成的任务直接打开文件,或右键选择打开文件/打开所在文件夹。
